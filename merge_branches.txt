## 5. –ö–∞–∫ –ø—Ä–æ–≤–µ—Ä–∏—Ç—å —Å–æ—Å—Ç–æ—è–Ω–∏–µ —Ä–µ–ø–æ–∑–∏—Ç–æ—Ä–∏—è

–ò—Å–ø–æ–ª—å–∑—É–π—Ç–µ –∫–æ–º–∞–Ω–¥—É *git status*, —á—Ç–æ–±—ã –ø—Ä–æ–≤–µ—Ä–∏—Ç—å —Ç–µ–∫—É—â–µ–µ —Å–æ—Å—Ç–æ—è–Ω–∏–µ —Ä–µ–ø–æ–∑–∏—Ç–æ—Ä–∏—è.

**–í—ã–ø–æ–ª–Ω–∏—Ç–µ:**

–ö–æ–º–∞–Ω–¥–∞ –ø—Ä–æ–≤–µ—Ä–∫–∏ —Å–æ—Å—Ç–æ—è–Ω–∏—è —Å–æ–æ–±—â–∏—Ç, —á—Ç–æ –∫–æ–º–º–∏—Ç–∏—Ç—å –Ω–µ—á–µ–≥–æ. –≠—Ç–æ –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ –≤ —Ä–µ–ø–æ–∑–∏—Ç–æ—Ä–∏–∏ —Ö—Ä–∞–Ω–∏—Ç—Å—è —Ç–µ–∫—É—â–µ–µ —Å–æ—Å—Ç–æ—è–Ω–∏–µ —Ä–∞–±–æ—á–µ–≥–æ –∫–∞—Ç–∞–ª–æ–≥–∞, –∏ –Ω–µ—Ç –Ω–∏–∫–∞–∫–∏—Ö –∏–∑–º–µ–Ω–µ–Ω–∏–π, –æ–∂–∏–¥–∞—é—â–∏—Ö –∑–∞–ø–∏—Å–∏.

## 6. –î–æ–±–∞–≤–ª–µ–Ω–∏–µ –∏–∑–º–µ–Ω–µ–Ω–∏–π

–ï—Å–ª–∏ –≤—ã –∏–∑–º–µ–Ω–∏–ª–∏ —Ñ–∞–π–ª—ã –∏–ª–∏ –¥–æ–±–∞–≤–∏–ª–∏ –Ω–æ–≤—ã–µ —Ñ–∞–π–ª—ã, —Ç–æ –Ω–∞–±—Ä–∞–≤ –∫–æ–º–∞–Ω–¥—É *git status* –≤—ã –º–æ–∂–µ—Ç–µ —É–≤–∏–¥–∏—Ç—å —Å–ª–µ–¥—É—é—â–∏–π —Ä–µ–∑—É–ª—å—Ç–∞—Ç:  

      git status  
      On branch master  
      Changes not staged for commit:  
      (use "git add <file>..." to update what will be committed)  
      (use "git checkout -- <file>..." to discard changes in working directory)  
      
      modified:   hello.html  
      
      no changes added to commit (use "git add" and/or "git commit -a")"

–ü–µ—Ä–≤–æ–µ, —á—Ç–æ –Ω—É–∂–Ω–æ –∑–∞–º–µ—Ç–∏—Ç—å, —ç—Ç–æ —Ç–æ, —á—Ç–æ **git** –∑–Ω–∞–µ—Ç, —á—Ç–æ —Ñ–∞–π–ª *hello.html* –±—ã–ª –∏–∑–º–µ–Ω–µ–Ω, –Ω–æ –ø—Ä–∏ —ç—Ç–æ–º —ç—Ç–∏ –∏–∑–º–µ–Ω–µ–Ω–∏—è –µ—â–µ –Ω–µ –∑–∞—Ñ–∏–∫—Å–∏—Ä–æ–≤–∞–Ω—ã –≤ —Ä–µ–ø–æ–∑–∏—Ç–æ—Ä–∏–∏.

–¢–∞–∫–∂–µ –æ–±—Ä–∞—Ç–∏—Ç–µ –≤–Ω–∏–º–∞–Ω–∏–µ –Ω–∞ —Ç–æ, —á—Ç–æ —Å–æ–æ–±—â–µ–Ω–∏–µ –æ —Å–æ—Å—Ç–æ—è–Ω–∏–∏ –¥–∞–µ—Ç –≤–∞–º –ø–æ–¥—Å–∫–∞–∑–∫—É –æ —Ç–æ–º, —á—Ç–æ –Ω—É–∂–Ω–æ –¥–µ–ª–∞—Ç—å –¥–∞–ª—å—à–µ. –ï—Å–ª–∏ –≤—ã —Ö–æ—Ç–∏—Ç–µ –¥–æ–±–∞–≤–∏—Ç—å —ç—Ç–∏ –∏–∑–º–µ–Ω–µ–Ω–∏—è –≤ —Ä–µ–ø–æ–∑–∏—Ç–æ—Ä–∏–π, –∏—Å–ø–æ–ª—å–∑—É–π—Ç–µ –∫–æ–º–∞–Ω–¥—É *"git add"*. –í –ø—Ä–æ—Ç–∏–≤–Ω–æ–º —Å–ª—É—á–∞–µ –∏—Å–ø–æ–ª—å–∑—É–π—Ç–µ –∫–æ–º–∞–Ω–¥—É *"git —Åheckout"* –¥–ª—è –æ—Ç–º–µ–Ω—ã –∏–∑–º–µ–Ω–µ–Ω–∏–π.

**–î–æ–±–∞–≤–∏–º –∏–∑–º–µ–Ω–µ–Ω–∏—è**

–î–∞–¥–∏–º –∫–æ–º–∞–Ω–¥—É **git** –ø—Ä–æ–∏–Ω–¥–µ–∫—Å–∏—Ä–æ–≤–∞—Ç—å –∏–∑–º–µ–Ω–µ–Ω–∏—è.

***–í—ã–ø–æ–ª–Ω–∏—Ç—å:***

`git add hello.html`

–ò–∑–º–µ–Ω–µ–Ω–∏—è —Ñ–∞–π–ª–∞ *hello.html* –±—ã–ª–∏ –ø—Ä–æ–∏–Ω–¥–µ–∫—Å–∏—Ä–æ–≤–∞–Ω—ã. –≠—Ç–æ –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ **git** —Ç–µ–ø–µ—Ä—å –∑–Ω–∞–µ—Ç –æ–± –∏–∑–º–µ–Ω–µ–Ω–∏–∏, –Ω–æ –∏–∑–º–µ–Ω–µ–Ω–∏–µ –ø–æ–∫–∞ –Ω–µ –ø–µ—Ä–º–∞–Ω–µ–Ω—Ç–Ω–æ (—á–∏—Ç–∞–π, –Ω–∞–≤—Å–µ–≥–¥–∞) –∑–∞–ø–∏—Å–∞–Ω–æ –≤ —Ä–µ–ø–æ–∑–∏—Ç–æ—Ä–∏–π. –°–ª–µ–¥—É—é—â–∏–π –∫–æ–º–º–∏—Ç –±—É–¥–µ—Ç –≤–∫–ª—é—á–∞—Ç—å –≤ —Å–µ–±—è –ø—Ä–æ–∏–Ω–¥–µ–∫—Å–∏—Ä–æ–≤–∞–Ω–Ω—ã–µ –∏–∑–º–µ–Ω–µ–Ω–∏—è.

***–ï—Å–ª–∏ –≤—ã —Ä–µ—à–∏–ª–∏, —á—Ç–æ –Ω–µ —Ö–æ—Ç–∏—Ç–µ –∫–æ–º–º–∏—Ç–∏—Ç—å –∏–∑–º–µ–Ω–µ–Ω–∏—è, –∫–æ–º–∞–Ω–¥–∞ —Å–æ—Å—Ç–æ—è–Ω–∏—è –Ω–∞–ø–æ–º–Ω–∏—Ç –≤–∞–º –æ —Ç–æ–º, —á—Ç–æ —Å –ø–æ–º–æ—â—å—é –∫–æ–º–∞–Ω–¥—ã **"git reset"** –º–æ–∂–Ω–æ —Å–Ω—è—Ç—å –∏–Ω–¥–µ–∫—Å–∞—Ü–∏—é —ç—Ç–∏—Ö –∏–∑–º–µ–Ω–µ–Ω–∏–π.***

–û—Ç–¥–µ–ª—å–Ω—ã–π —à–∞–≥ –∏–Ω–¥–µ–∫—Å–∞—Ü–∏–∏ –≤ **git** –ø–æ–∑–≤–æ–ª—è–µ—Ç –≤–∞–º –ø—Ä–æ–¥–æ–ª–∂–∞—Ç—å –≤–Ω–æ—Å–∏—Ç—å –∏–∑–º–µ–Ω–µ–Ω–∏—è –≤ —Ä–∞–±–æ—á–∏–π –∫–∞—Ç–∞–ª–æ–≥, –∞ –∑–∞—Ç–µ–º, –≤ –º–æ–º–µ–Ω—Ç, –∫–æ–≥–¥–∞ –≤—ã –∑–∞—Ö–æ—Ç–∏—Ç–µ –≤–∑–∞–∏–º–æ–¥–µ–π—Å—Ç–≤–æ–≤–∞—Ç—å —Å –≤–µ—Ä—Å–∏–æ–Ω–Ω—ã–º –∫–æ–Ω—Ç—Ä–æ–ª–µ–º, **git** –ø–æ–∑–≤–æ–ª–∏—Ç –∑–∞–ø–∏—Å–∞—Ç—å –∏–∑–º–µ–Ω–µ–Ω–∏—è –≤ –º–∞–ª—ã—Ö –∫–æ–º–º–∏—Ç–∞—Ö, –∫–æ—Ç–æ—Ä—ã–µ —Ñ–∏–∫—Å–∏—Ä—É—é—Ç —Ç–æ, —á—Ç–æ –≤—ã —Å–¥–µ–ª–∞–ª–∏.

–ü—Ä–µ–¥–ø–æ–ª–æ–∂–∏–º, —á—Ç–æ –≤—ã –æ—Ç—Ä–µ–¥–∞–∫—Ç–∏—Ä–æ–≤–∞–ª–∏ —Ç—Ä–∏ —Ñ–∞–π–ª–∞ *(a.html, b.html, –∏ c.html)*. –¢–µ–ø–µ—Ä—å –≤—ã —Ö–æ—Ç–∏—Ç–µ –∑–∞–∫–æ–º–º–∏—Ç–∏—Ç—å –≤—Å–µ –∏–∑–º–µ–Ω–µ–Ω–∏—è, –ø—Ä–∏ —ç—Ç–æ–º —á—Ç–æ–±—ã –∏–∑–º–µ–Ω–µ–Ω–∏—è –≤ *"a.html"* –∏ *"b.html"* –±—ã–ª–∏ –æ–¥–Ω–∏–º –∫–æ–º–º–∏—Ç–æ–º, –≤ —Ç–æ –≤—Ä–µ–º—è –∫–∞–∫ –∏–∑–º–µ–Ω–µ–Ω–∏—è –≤ *"c.html"* –ª–æ–≥–∏—á–µ—Å–∫–∏ –Ω–µ —Å–≤—è–∑–∞–Ω—ã —Å –ø–µ—Ä–≤—ã–º–∏ –¥–≤—É–º—è —Ñ–∞–π–ª–∞–º–∏ –∏ –¥–æ–ª–∂–Ω—ã –∏–¥—Ç–∏ –æ—Ç–¥–µ–ª—å–Ω—ã–º –∫–æ–º–º–∏—Ç–æ–º.

–í —Ç–µ–æ—Ä–∏–∏, –≤—ã –º–æ–∂–µ—Ç–µ —Å–¥–µ–ª–∞—Ç—å —Å–ª–µ–¥—É—é—â–µ–µ:
